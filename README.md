# TwentyOnePoints
Game twenty one points

Compile:

git clone https://github.com/HYSM-SM/TwentyOnePoints

cd TwentyOnePoints

chmod +x compile.sh

./compile.sh

Compile in Termux(clang-15):

git clone https://github.com/HYSM-SM/TwentyOnePoints

cd TwentyOnePoints

chmod +x compileTermux.sh

./compileTermux.sh
